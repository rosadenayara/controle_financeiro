# 📖 Sobre

O Controle Real é uma plataforma desenvolvida para auxiliar usuários no gerenciamento completo da vida financeira.

O sistema reúne ferramentas de controle financeiro, planejamento de investimentos, cálculo de impostos, acompanhamento de metas e dashboards inteligentes para apoiar a tomada de decisões financeiras.

Além do gerenciamento de receitas e despesas, a aplicação oferece análises patrimoniais, comparativos de investimentos e indicadores econômicos atualizados.

# 🚀 Executando localmente

Clone o projeto

bash
git clone https://github.com/rosadenayara/controle_financeiro.git


Entre na pasta

bash
cd controle_financeiro


Crie um ambiente virtual

bash
python -m venv venv


Ative o ambiente

Windows

bash
venv\Scripts\activate


Linux

bash
source venv/bin/activate


Instale as dependências

bash
pip install -r requirements.txt


Execute

bash
python run.py
